# My [waybar](https://github.com/Alexays/Waybar) configuration files


### Dependencies:
```
sudo pacman -S cantarell-fonts ttf-hack-nerd
```

### Installation:
```
cd
mkdir config
cd ./config/
git clone git@github.com:jszbm/waybar.git
cd ..
cp -r ./config/waybar ./.config/waybar/
cd ./.config/waybar
```

Laptop:
```
cp config-laptop.jsonc config.jsonc
```

Desktop:
```
cp config-desktop.jsonc config.jsonc
```
